HTML Review

HTML é uma linguagem de Marcação de textos, utilizada para estruturar a pagina web
é constituida por < TAGs >, < Elementos > das tags e < Atributos >
TAGs não são Case Sensitive

Tags iniciais

< !DOCTYPE html > - o tipo do documento para reconhecimento do browser

< HTML lang="pt-BR"> < /html>  'language' linguagem reconhecida pelos apps que utilizarem o site

< head> < / head >   cabeçalho sobre o site em si

  < meta>  - Metadados sobre o site que não aparecem para o usuário inicialmente

       elementos: charset="UTF-8"; name="description" content="...descrição do site...";

  < Title> < / Title >- Titulo da pagina que ficara a mostra na Aba do navegador

  < link> - link para outros arquivos
        arquivo CSS ( < rel="stylesheet" href="style.css"> )
        Fav icon ( rel="icon" href="favicon.ico" type="image/x-icon" /> )
        TARGET="" especifica onde abrir o documento vinculado.
         _self - Padrão. na mesma janela/guia em que foi clicado
         _blank - Nova janela ou guia
        To email: < a href="mailto:someone@example.com" >
Botão como link: < button onclick="document.location='default.asp'"> Exmemple < / button >

O elemento < s cript> é utlizado para linkar programas em JS no texto html

< body> < /body> - Corpo do texto. O que efetivamente aparece na pagina web

A tag < h1 > < /h1 > define um título/cabeçalho grande - vai de 1 a 6

A tag < p > < /p > define um parágrafo
   elementos de 'p': title="tooltip"

A tag < br > define uma quebra de pagina "enter"

A tag < hr > define uma linha horizontal

A tag < a > < /a > define um link/ancora
   elementos de "a": href="www..." = Hiperlink Reference

A tag < pre> exibe um texto pre formatado, com fonte menor e com auto quebra

A tag < img > define uma imagem
   elementos de 'img': src="link"; alt="descrição-da-img" ;c/ style="" width ="largura" ; height="altura"
   src= source / indica o caminho do arquivo (path) // Absolutos (links externos) Relativos (links internos)

   < map > < /map > - imagem com areas clicáveis, < area >; < usemap >;

A tag < picture > permite mostrar imagens diferente em aparelhos diferentes
           (+ Largura de banda e formato do arquivo)

Elemento < div> -ocupa todo o block (full width); guarda outros elementos HTML; divisão/seção

Elemento < span> - utiliza apenas o espaço necessário, só parte do texto
