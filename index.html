HTML Review

HTML é uma linguagem de Marcação de textos, utilizada para estruturar a pagina web
é constituida por < TAGs >, < Elementos > das tags e < Atributos >
TAGs não são Case Sensitive

Tags iniciais

"< !DOCTYPE html" > - o tipo do documento para reconhecimento do browser

< HTML lang="pt-BR"> < /html>  'language' linguagem reconhecida pelos apps que utilizarem o site

< head > < / head >   cabeçalho sobre o site em si

 < meta >  - Metadados sobre o site que não aparecem para o usuário inicialmente

       elementos: charset="UTF-8"; name="description" content="...descrição do site...";

  < Title > - Titulo da pagina que ficara a mostra na Aba do navegador

< body > < /body > - Corpo do texto. O que efetivamente aparece na pagina web

A tag < h1 > < /h1 > define um título grande - vai de 1 a 6

A tag < p > < /p > define um parágrafo
   elementos de 'p': title="tooltip"

A tag < br > define uma quebra de pagina "enter"

A tag < hr > define uma linha horizontal

A tag < a > < /a > define um link
   elementos de "a": href="www..." = Hiperlink Reference

A tag < img > define uma imagem
   elementos de 'img': src="link"; alt="descrição-da-img" ; width ="largura" ; height="altura"
   src= source / indica o caminho do arquivo (path) // Absolutos (links externos) Relativos (links internos)

A tag < pre > exibe um texto pre formatado, com fonte menor e com auto quebra





















Observações

  • Dois espaços — Digite:  & ensp;
  • Quatro espaços — Digite:  & emsp;
  • Recuo — Digite:  & nbsp;& nbsp;& nbsp;& nbsp;

Elementos comuns: style="color:red"/

  • Use background-color for background color
  • Use color for text colors
  • Use font-family for text fonts
  • Use font-size for text sizes
  • Use text-align for text alignment
